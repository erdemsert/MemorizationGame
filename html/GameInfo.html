Game Information

The game rules are simple. There are four different numbers which is generated by random number between 0-100 and each number is modified as unique in order words there is no number same as the other. I want to you to find out the numbers by clicking the box and each box contains a rnd number You have to guess it. After finding the 4 number the screen changes and writes you finished it.
